# Lesson 11 & 12: Final Project

## Overview
Build an interactive website that showcases what you've learned!

## Requirements

### Must Have
- [ ] **HTML**: Proper structure with semantic elements
- [ ] **CSS**: Custom styling (not just defaults)
- [ ] **JavaScript**: At least one interactive feature
- [ ] **Multi-section**: Navigation and multiple content areas

### Interactive Feature Ideas (pick at least one)
- Dark mode toggle
- Show/hide content (accordion, modal, tabs)
- Form with validation
- Counter or calculator
- Image gallery
- Quiz or game

## Project Ideas

### 1. Personal Portfolio
- About section
- Skills showcase
- Project gallery with interactive features
- Contact section

### 2. Interactive Quiz
- Multiple choice questions
- Score tracking
- Results display
- Option to restart

### 3. Recipe Site
- Recipe cards with ingredients
- Expandable instructions
- Serving size calculator
- Favorite/bookmark feature

### 4. Mini Game
- Simple game (number guessing, memory match, rock-paper-scissors)
- Score tracking
- Play again functionality

### 5. Product Landing Page
- Hero section
- Feature highlights
- FAQ accordion
- Theme switcher

## Tips

- **Start with HTML structure** before styling
- **Work section by section** - don't try to do everything at once
- **Test frequently** - check in browser after each change
- **Copy patterns** from previous lessons
- **Ask for help** when stuck!
