# Web Development Intro Course

A 6-week sampler course covering HTML, CSS, and JavaScript fundamentals.

## Getting Started in GitHub Codespaces

1. Click the green **Code** button on this repo
2. Select **Codespaces** → **Create codespace on main**
3. Wait for the environment to build (first time takes a minute)

## Viewing Your HTML Pages

This Codespace comes with **Live Server** pre-installed.

**To preview any HTML file:**
1. Right-click the HTML file in the file explorer
2. Select **Open with Live Server**
3. A browser preview will open automatically

**Or:** Click **Go Live** in the bottom-right status bar.

## Tips for Students

- Save your files often (Cmd/Ctrl + S)
- Check the browser preview after each change
- Use the browser's Developer Tools (F12) to debug
- Don't worry about memorizing everything—look things up!
