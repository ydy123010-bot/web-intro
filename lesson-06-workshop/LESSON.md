# Lesson 6: Workshop - Build a Page

## Goals
- Apply HTML and CSS skills to build a complete page
- Practice problem-solving and debugging
- Get creative!

## Project Requirements

Build a single-page website with:

1. **Navigation** - Links to different sections
2. **Hero/Header** - Eye-catching intro section
3. **Content Sections** - At least 2-3 sections
4. **Footer** - Copyright, links, or contact info
5. **Styling** - Custom colors, fonts, spacing

## Project Ideas

### Portfolio
- About me section
- Skills list
- Projects/work showcase
- Contact info

### Fan Page
- Hero with main image
- Info/facts section
- Gallery of images
- Links to related content

### Product Landing
- Product hero image
- Features list
- Testimonials
- Call to action

## Tips

- **Start with HTML structure first**, then add CSS
- **Work section by section** - don't try to do everything at once
- **Use DevTools** (F12) to debug layout issues
- **Copy patterns** from lesson-05 as a starting point
- **Ask for help** when stuck!
