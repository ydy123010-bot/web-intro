# Lesson 2: HTML Content Elements

## Goals
- Use headings, paragraphs, lists, and links
- Add images to a page
- Understand semantic HTML

## Key Concepts

### Links
```html
<a href="https://google.com">Go to Google</a>
<a href="other-page.html">Another page</a>
```

### Images
```html
<img src="photo.jpg" alt="Description of image">
<img src="https://example.com/image.png" alt="External image">
```

### Lists
```html
<!-- Unordered (bullets) -->
<ul>
  <li>Item one</li>
  <li>Item two</li>
</ul>

<!-- Ordered (numbers) -->
<ol>
  <li>First step</li>
  <li>Second step</li>
</ol>
```

### Semantic Elements
These describe the meaning of content:
- `<header>` - Top of page/section
- `<main>` - Main content
- `<section>` - Grouped content
- `<footer>` - Bottom of page/section
