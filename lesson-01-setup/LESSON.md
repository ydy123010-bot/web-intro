# Lesson 1: Setup & First Page

## Goals
- Understand what HTML is
- Create your first HTML file
- View it in the browser

## Key Concepts

### What is HTML?
HTML (HyperText Markup Language) tells the browser what content to display. It uses **tags** to mark up content.

### Basic Structure
```html
<!DOCTYPE html>        <!-- Tells browser this is HTML5 -->
<html lang="en">       <!-- Root element -->
<head>                 <!-- Metadata (not visible) -->
  <title>Page Title</title>
</head>
<body>                 <!-- Visible content goes here -->
  <h1>Hello!</h1>
</body>
</html>
```

### Common Tags
- `<h1>` to `<h6>` - Headings (h1 is biggest)
- `<p>` - Paragraph
- `<br>` - Line break
